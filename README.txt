1. "Add/Update" button performs both mentioned tasks. Initially it adds the form's contents in a new row in the table. Additionally,
    if it used after the "Edit" button of a row, if replaces that row's values with the current values of the form.
2. Both "Edit" and "Delete" options appear on the side of every row. The "Edit" button transfers the row's values in the form, where they 
    can be changed and stored as its new values with "Add/Update". The "Delete" button deletes the whole row of the table.
